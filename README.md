# Doom On Album Art

This is DOOM, you can play it using your system's hardware media keys with the Album art thingy on windows (appearently it is called Widnows Global Media Overlay) You can open it with `Win + A`.



##  Controls

| Media Key | Action |  |
| :--- | :--- | :--- |
| **Previous Track** | Toggle Run Left | Holds `Arrow Up` + `Arrow Left` |
| **Next Track** | Toggle Run Right | Holds `Arrow Up` + `Arrow Right` |
| **Play / Pause** | Action/Combat | Presses `S` (Shoot), `W` (Open) and `Enter` |
